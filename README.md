# ShiroShare

预想的功能是把分享的文件夹同步到其他电脑上，其他电脑上的人要是修改的话还可以同步到本地，有点类似于微软的脱机文件吧，不过我想实现的的功能主要是往局域网网盘的方向靠近，局域网内大家都是主机，可以保证数据不会丢失，但是也会出现安全和储存问题，考虑到使用的环境，是在办公室内的局域网，分享的文件也多是文档类的文件，这些问题也可以暂时忽略了，
